# PDF → MD Converter

Browser-based converter for PDF, DOCX, and PPTX files to Markdown.

Hosted on GitHub Pages — no server, no uploads, files stay on your machine.

## Usage

1. Open the app URL in Chrome
2. Drop files or click to browse
3. Click **CONVERT & DOWNLOAD ZIP**
4. Open the downloaded `converted.zip`

## Limitations

- Scanned PDFs (image-based) produce empty output — no OCR
- Complex multi-column PDF layouts may have incorrect text order
- PPTX images and charts are skipped (text only)
